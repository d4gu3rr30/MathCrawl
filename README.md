## 📝 Описание

**MathCrawl** — это образовательная двухмерная игра в жанре roguelike с элементами процедурной генерации и адаптивной системой сложности математических задач. Приложение предназначено для эффективного развития вычислительных навыков у школьников, студентов и всех, кто хочет улучшить свои математические способности в игровой форме.

### Основные особенности

- 🎲 **Процедурная генерация уровней** — каждое прохождение уникально
- 🧠 **Адаптивная сложность** — система подстраивается под уровень игрока
- 📊 **Система мета-прогрессии** — накапливайте «Очки знаний» для перманентных улучшений
- 💾 **Локальное сохранение** — прогресс хранится в формате JSON

## 📦 Установка и запуск

### Вариант 1: Для пользователей (Готовый установщик от seqquo prod.)

1. Перейдите в раздел **Releases** в репозитории проекта на GitHub.
2. Скачайте файл установщика `MathCrawl_Setup.exe`.
3. Запустите инсталлятор и следуйте стандартным инструкциям мастера установки (доступно автоматическое создание ярлыков в меню «Пуск» и на Рабочем столе).
4. Ознакомьтесь и примите условия **Лицензионного соглашения конечного пользователя (EULA)**.
5. Запустите игру через ярлык.

### Вариант 2: Для разработчиков (Запуск из исходного кода)

> **Требование:** на вашем компьютере должен быть установлен интерпретатор **Python версии 3.10 или выше**.

1. Клонируйте репозиторий с проектом:

```bash
git clone https://github.com/d4gu3rr30/MathCrawl.git
cd MathCrawl
```

2. Установите необходимые зависимости (библиотеку Pygame):

```bash
pip install pygame
```

3. Запустите игру:

```bash
python main.py
```

## 🎮 Управление (Controls)

| Клавиша / Устройство | Действие |
|----------------------|----------|
| `W`, `A`, `S`, `D` | Перемещение персонажа |
| `Мышь` (движение) | Прицеливание |
| `Левая кнопка мыши` (ЛКМ) | Стрельба / Выбор цели с математическим ответом |
| `E` | Взаимодействие с объектами (сундуки, торговые терминалы) |
| `I` / `Tab` | Открыть/закрыть меню инвентаря и статистики |
| `Esc` | Пауза / Выход в главное меню |

## 📊 Архитектура данных и автономность

В проекте реализован паттерн структуры данных **Single Table Inheritance** (Единая таблица), адаптированный под JSON/классическое реляционное хранение. Сущности `Weapon` (Оружие) и `Artifact` (Артефакт) объединены в общую структуру `Item` со специфицирующим маркером `item_type`.

### Портативная архитектура (Portable)

Для обеспечения полной автономности приложение скомпилировано с помощью **PyInstaller**. Папка `_internal` содержит:

- Ядро интерпретатора (`python312.dll`)
- Системные библиотеки (`VCRUNTIME140.dll`)
- Скомпилированные расширения C (`.pyd`)
- Зависимости движка Pygame

Это изолирует игровое окружение и гарантирует работоспособность дистрибутива на целевых операционных системах без предварительной настройки среды Python.
